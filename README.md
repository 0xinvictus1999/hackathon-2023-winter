- 三等奖: 2127 DOT 约一万美金 (每个赛道各二个)
- 最受观众喜爱奖: 1063 DOT 约五千美金 （共一个）
- 最受开发者喜爱奖: 1063 DOT 约五千美金 （共一个）
- 最佳创新奖: 1063 DOT 约五千美金 （共一个）
- Travel Grant奖金池: 6382 DOT 约三万美金 （共一个）
（打卡每场活动+进入决赛来现场的战队共同瓜分该奖金池）
